# Solution 2: MIMIZUKU Dual Field Viewer (Side-by-Side)

## 概要 / Overview

このフォルダは、MIMIZUKU（ミミズク）デュアルフィールドビューワーの別実装例です。
ターゲット天体とガイド星の視野を左右に並べて表示します。

## 特徴 / Features

### MIMIZUKU表示方式: 左右分割ビュー (Side-by-Side View)

- ターゲットとガイド星を個別の画像で並べて表示
- 各フィールドを独立して拡大・操作可能
- より詳細な確認が可能
- 各フィールドの視野を個別に調整可能

### 主な機能

1. **天体検索**: 座標入力による天体検索
2. **カタログ読み込み**: CSV形式の星カタログ読み込み
3. **位置角（P.A.）制御**: 観測装置の位置角設定
4. **MIMIZUKU表示**: 左右分割でのデュアルフィールド可視化
5. **観測可能性チェック**: TAO等の観測所での観測可能性確認

## 使い方 / Usage

1. `index.html`をWebブラウザで開く
2. ターゲット座標を入力して検索
3. カタログを読み込み
4. ガイド星を選択
5. "Show MIMIZUKU Dual Field View"をクリック

## 技術仕様 / Technical Specifications

- **フロントエンド**: 純粋なJavaScript
- **天体画像**: Aladin Lite v3 (2インスタンス)
- **バックエンド**: Python Flask (ライトカーブデータ用)
- **データベース**: SQLite (NEOWISE データ)

## MIMIZUKU表示の実装 / MIMIZUKU Implementation

### Solution2の特徴

- **2つのAladinインスタンス**: 左右にターゲットとガイド星を個別表示
- **独立操作**: 各フィールドを独立して拡大・縮小可能
- **詳細確認**: 各天体周辺をより詳細に確認可能
- **視野サイズ**: 各フィールドで個別に最適化

### 利点

✓ 各フィールドを詳細に確認できる  
✓ 独立した操作が可能  
✓ 分離距離が大きい場合でも見やすい  
✓ 各天体周辺の星を個別に確認できる

### 用途

- 分離距離が大きい場合に最適
- 各フィールドを詳細に確認したい場合
- 最終確認や詳細観測計画での使用

### Solution1との違い

| 項目 | Solution1 | Solution2 |
|------|-----------|-----------|
| **表示方式** | 単一統合ビュー | 左右分割ビュー |
| **Aladinインスタンス** | 1個 | 2個 |
| **視野調整** | 統一 | 個別 |
| **詳細度** | 全体的 | 個別詳細 |
| **メモリ使用** | 少ない | やや多い |
| **最適用途** | 小分離距離 | 大分離距離 |
